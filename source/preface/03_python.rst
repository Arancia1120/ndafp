Biopython
---------

biopython_ 是 python_ 语言的专门生物信息程序包。

安装 biopython
^^^^^^^^^^^^^^

建议采用 virtualenv 建立虚拟环境来安装。

.. code-block:: bash

   ~$ sudo apt-get install virtualenv virtualenv_wrapper
   ~$ mkvirtualenv bio
   ~$ workon bio
   (bio)~$ pip install biopython numpy reportlab
